Interface IF_T100_DYN_MSG Dynamic Access to T100 Texts (with Attributes)

Main Package
Documentation

Meaning

Erweiterung des Interface IF_T100_MESSAGE um Attribute für den Nachrichtentyp und die Platzhalter der Nachricht.

Use

Das Interface IF_T100_DYN_MSG ist für Auslösen von klassenbasierten Ausnahmen mit dem Zusatz MESSAGE der Anweisung RAISE EXCEPTION vorgesehen. Die Eigenschaften der hinter MESSAGE angegebenen Nachricht werden bei einer Ausnahmeklasse, die IF_T100_DYN_MSG einbindet, automatisch den zugehörigen Attributen zugewiesen.
