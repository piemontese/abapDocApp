Interface IF_T100_MESSAGE Interface for Accessing T100 Texts

Main Package
Documentation

Meaning

Interface für Klassen, deren Instanzen mit T100-basierten Nachrichten assoziert sind, deren sprachunabhängige Charakterisierung im Attribut T100KEY zu finden sind. Diese Instanzen können bei der Anweisungsvariante MESSAGE oref TYPE mtype verwendet werden. Üblicherweise wird dieses Interface von Ausnahmeklassen implementiert, bei denen es sinnvoll ist, die assozierten Fehlernachrichten direkt an den Benutzer weiterzugeben.

Das Attribut T100_KEY enthält die Nachrichtenklasse, die Nachrichtennummer, sowie Namen der Attribute, deren Werte in den Textplatzhaltern verwendet werden. Dabei kann nicht nur ein einfacher Attributname verwendet werden, sondern ein beliebiger Ausdruck, der Offset-/Längenangaben, Strukturkomponenten-Selektoren und Komponentenselektoren enthält (wie es bei der dynamischen Angabe im ASSIGN möglich ist).

Dieses Interface enthält das Interface IF_MESSAGE ,  d.h., Klassen, die das Interface implementieren, müssen die Methoden GET_TEXT und GET_LONGTEXT bereitstellen, mit denen der Nachrichtentext abgerufen werden kann (in Kurz- und Langform).

Hints

Das Interface IF_T100_DYN_MSG erweitert IF_T100_MESSAGE um Attribute für den Nachrichtentyp und die Platzhalter der Nachricht.
